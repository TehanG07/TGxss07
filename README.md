# TGxss07

**TGxss07** is a tool for finding XSS vulnerabilities in web applications.

## Usage

single url

python TGxss07.py -u http://example.com

multiple url

python TGxss07.py -l urls.txt

multiple domains/sub-domains

python TGxss07.py -dL domains.txt
